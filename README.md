# Coding Skills

A Claude Code plugin with coding convention skills for Kotlin, Android, and Compose Multiplatform.

## Installation

```
/plugin marketplace add necatisozer/coding-skills
/plugin install coding-skills
```

## Skills

| Skill | Description |
|---|---|
| `kotlin-conventions` | File organization, idioms, type design, Duration/Instant APIs |
| `kotlin-coroutines-conventions` | Coroutine safety, Flow patterns, runCatching, dispatcher injection |
| `kotlin-serialization-conventions` | kotlinx.serialization response/request models, value classes for API enums |
| `compose-conventions` | Modifiers, state management, lazy layouts, lifecycle effects, UDF, insets |
| `compose-resource-conventions` | Image formats, icon naming/sizing, string resources, PNG/SVG conversion |
| `android-conventions` | Platform API compatibility, runtime pitfalls |
| `jira-conventions` | Jira issue handling with comment fetching |
| `figma-conventions` | Design token resolution via MCP |
| `slack-conventions` | Slack message formatting for MCP tools |
| `gws-conventions` | Google Workspace CLI usage |
